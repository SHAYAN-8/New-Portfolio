RESTAURANT APP

MS Foods Restaurant App

This project is a simple frontend restaurant app built with HTML, CSS, Tailwind CSS, JavaScript, React.js with Context API. User can add items to the cart and applying vouchers for discounts The project also utilizes additional libraries like React Router for navigation, React Icons, React Hot Toast for a great user experience.



Main Features


  • Navigation:

    Implements React Router for smooth and dynamic navigation.


  • Attractive Icons:

    Integrates React Icons to create an awesome user-interface experience.


  • Animated Toast:

    This project incorporates React Hot Toast for interactive toast notifications.
